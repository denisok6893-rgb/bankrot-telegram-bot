Поддерживается отображение кодов в юридический текст:

- `married`  → `Состоит в зарегистрированном браке.`

- `divorced` → `Брак расторгнут.`

Если в карточке уже хранится русский текст — используется как есть.  
Если поле пустое — `"не указано"`.

## 2025-12-29 — Меню документов по делу (последний + архив)

### Что сделано
- Приведено в порядок меню «Документы по делу»
- Реализовано разделение:
  - «Последний документ» — открывает самый свежий .docx
  - «Архив документов» — все ранее сгенерированные документы
- Убрана передача имени файла через callback_data
- Архив и последний документ берутся напрямую из файловой системы

### Почему так
- callback_data в Telegram ограничен по длине
- длинные имена файлов обрезались → «Файл не найден»
- файловая система — единственный надёжный источник истины
- архитектура теперь масштабируется под новые типы документов

### Текущее состояние

- Архив документов листается и файлы скачиваются
- Изменения зафиксированы в Git (commit: 2c95ac2)

### Как проверить
1. Профиль → Дела → открыть дело
2. Документы по делу
3. Нажать «Последний документ»
4. Зайти в «Архив документов» и открыть любой файл
