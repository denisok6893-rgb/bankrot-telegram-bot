# Bankrot Telegram Bot

Telegram-бот для помощи юристу по банкротству физических лиц (РФ).

## Цель проекта
Автоматизация подготовки процессуальных документов по ФЗ-127
на основе карточки дела клиента.

Проект разрабатывается с прицелом на коммерческое использование.

## Текущий статус
- Бот работает (aiogram)
- Реализована базовая логика дел
- В работе основной документ:
  "Заявление о признании гражданина банкротом"

## Ключевые архитектурные решения
- Один источник данных — карточка дела
- Документы формируются ТОЛЬКО по DOCX-шаблонам
- ФИО полностью указывается только в шапке документа,
  далее используется формат "Фамилия И.О."
- Семейное положение, дети, имущество — динамические блоки
- Документы не генерируются без выбранного дела

## Безопасность
- Персональные данные не логируются
- Секреты хранятся в .env (см. .env.example)
- Доступ к боту ограничен

## Что сознательно НЕ делаем
- Не даём юридических прогнозов
- Не генерируем текст "с нуля"
- Не принимаем решения за юриста

## Ближайшие шаги
- Получить реальные заполненные заявления от юриста
- Проверить корректность контракта данных
- После этого реализовать генерацию документа через Codex
